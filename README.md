# triangle-points-C
Program reads points from file and if points can create triangle, calculates: area,circumference and output it to text file
## Arguments
program takes two arguments input_path and output_path.
### Input
Input shuld be formated like this:
```
Xa Ya Xb Yb Xc Yc
```
Points must be inatgers.
check example_input.txt
